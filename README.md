# Bang & Olufsen

## Technologies used
- **BEM Methodology**
- **SCSS Preprocessor**
- **CSS Grid**
- **CSS Flexbox**

## Responsive layout
The responsive layout is built using CSS Grid and Flexbox.
It is optimized for the following devices:
- Mobile
- Tablet
- Laptop
- Large screen
