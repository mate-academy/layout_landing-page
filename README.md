 PROJECT OVERVIEW
This project is a responsive landing page.
It is optimized for various devices, ensuring a consistent look and feel on desktop, tablet, and mobile screens.

TECHNOLOGIES USED
HTML5;
CSS3 (including Flexbox and Grid);
Google fonts (for typography)

Additional Instructions for Launch

    Clone the repository to your local machine:

   1. git clone <https://github.com/AndriiKasiian/layout_landing-page.git>;

   2.Navigate to the project directory:

    "cd layout_landing-page" ;

    3.Run "npm i" in Terminal ;

    4. Run "npm start" in Terminal .
