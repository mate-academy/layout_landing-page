## 🚀 Getting Started

### 1. Clone the repository:

```bash
git clone https://github.com/yourusername/landing-page.git
```

### 2. Install dependencies:

```bash
npm install
```

### 3. Start the development server:

```bash
npm run start
```

### 4. Open the project in your browser:

http://localhost:1234
